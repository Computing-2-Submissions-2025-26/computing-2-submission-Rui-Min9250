CatCakeKitchen

Namespace

CatCakeKitchen

Description:

  • Cat Cake Kitchen is a module to model the state and rules of a turn-based, board-based kitchen collection game. The module stores no browser state; it only describes the game board, cat chef, cake orders, ingredients, obstacles, movement, collection, and win or loss conditions. This means a game can be simulated from the console or from unit tests without using the web page.
